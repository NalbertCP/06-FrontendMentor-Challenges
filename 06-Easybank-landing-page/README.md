## 🎮 Iniciando o projeto

1. Clone este repositório;
2. Baixe as dependências deste projeto com npm install;
3. Inicie a aplicação com npm run dev;
4. Navegue até http://localhost:3000/ para visualizar o projeto.

## 💻 Stack utilizada

-   HTML
-   CSS
-   JavaScript
-   ReactJS
-   TailwindCSS
